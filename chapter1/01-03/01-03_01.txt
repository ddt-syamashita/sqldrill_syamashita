問1
アンケート回答テーブル（quest）から都道府県名を重複のない形式で取り出してみましょう。以下の空欄を埋めてSQL命令を完成させてください。


SELECT ①
	prefecture
FROM
	②
;

指定した列を重複の値を除いて選択する文
①DISTINCT
アンケート回答テーブルから情報を取得する
②quest

実行結果
mysql> SELECT DISTINCT
    -> prefecture
    -> FROM
    -> quest;
+------------+
| prefecture |
+------------+
| 東京都     |
| 神奈川県   |
| 茨城県     |
| 千葉県     |
| 埼玉県     |
+------------+
5 rows in set (0.00 sec)
